Introduction to HTML and CSS

Html & CSS are used to design and build websites. All websites use HTML and CSS. Html stands for Hyper Text Mark-up Language. It is use for creating web pages and web applications.

CSS stands for Cascading Style Sheet. CSS describes how Html elements are to be displayed on the screen, paper or in other media.

Web Applications

In computing, a web application or web app is a client-server computer program in which the client (including the user interface and client-side logic) runs in a web browser.

Web Pages are what make up a website. For example on www.teampiccolo.com, you can find the home page, about us page, services page, portfolio page and contact us page.

A Website is a collection of related web pages located under a single domain name. When you combine the home page, about us page, services page, portfolio page, contact us page and store them under a single domain name www.teampiccolo.com

A web server is a computer that runs websites.

Domain Name and Web Hosting

Domain name is easy-to-remember words that we can use to communicate to a DNS (Domain Name System) server, the website we want to visit. For example www.teampiccolo.com, www.thepyramid.online etc.

Note: DNS (Domain Name System) or (Domain Name Service) or (Domain Name Server) they are all the same.

Web Hosting is the technologies and services needed for a website or web page to be viewed on the internet.

Note: after designing your website, you need to buy a domain name on the internet and a hosting package. The price varies depending on the hosting company and the type of package you are requesting.

Margin Property

The Margin-Property is divided into four:
  • margin-top
  • margin-bottom
  • margin-right
  • margin-left
body{
   margin-top: 5px;
   margin-bottom: 15px;
   margin-right: auto
   margin-left: auto
}

Text Align Property

The text-align property allows you to control the alignment of text.

The property can take one of the four values:

  • left
  • right
  • center
  • justify

#container{
   text-align: justify;
}

Float Property

The float property positions the columns next to each other.

The property can take one of the two values:

  • left: this indicates that the column should float to the left.
  • right: this indicates that the column should float to the right.

nav{
   float: right;
}

Text Decoration Property

The property can take one of the four values:

  • none: this removes any decoration already applied to the text.
  • underline: this adds a line underneath the text.
  • overline: this adds a line over the top of the text.
  • line-through: this adds a line through words.

a{
   text-decoration: none;
}
